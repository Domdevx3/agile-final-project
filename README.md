# agile-final-project
**Coursera IBM Introduction to Agile Development and Scrum Project**

**On this lab the simulation was about creating a project (Kanban board) to manage the tasks for each sprint**

Agile artifacts 
## 1. User Stories & Acceptance Criteria
User stories were drafted folling the standard format:
'As a [type of user], I want [need/action] so that [benefit/value]'
* **Example User Story:**
  * **As a** registered buyer, **I want** to save items to a shopping cart **so that** I can review and purchase them later.
  * **Acceptance Criteria:**
    * Users can add and remove items from the cart.
    * Cart data persists if the user logs out.

## 2. Product Backlog & Grooming
* Defined the **Definition of Done DoD**.
* Estimated effort using **Story Points**.
* Prioritized stories based on business value.

## 3.Sprint Planning
* **Sprint 1 (2 weeks):** Focus on user authentication and core product catalog.
* **Sprint 2 (2 weeks):** Focus on shopping cart functionality and payment gateway integration.
